# Quick Emigrate — Q_E

Plataforma digital de asistencia migratoria para hispanoamericanos que quieren emigrar a España.

## Estructura
- `/frontend` — Landing page en React + Vite + TypeScript
- `/backend` — API REST en Node.js + Express + TypeScript

## Desarrollo local

### Frontend
cd frontend
npm install
npm run dev

### Backend
cd backend
npm install
cp .env.example .env
npm run dev
